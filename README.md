# Customer Churn Prediction

## Project Overview
This project is an end-to-end supervised machine learning solution for predicting customer churn using the dataset `customer_churn_data.csv`. It is designed to be submission-ready for GitHub and the Thiranex internship task, "Predictive Modeling Using Machine Learning".

## Objective
Build a professional churn prediction workflow that:
- cleans and prepares the data automatically,
- trains multiple supervised learning models,
- compares their performance using appropriate metrics,
- visualizes the results,
- and saves all deliverables in a clean project structure.

## Dataset
- File: `customer_churn_data.csv`
- Rows: 5,880
- Columns: 21
- Target column: `Churn`
- Identifier column removed during preprocessing: `customerID`

The dataset contains customer demographics, contract information, service usage, billing values, and churn labels.

## Libraries Used
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- jupyter

## Workflow
1. Load the dataset.
2. Inspect shape, columns, data types, missing values, duplicates, and summary statistics.
3. Clean the dataset by removing duplicates and dropping identifier columns.
4. Encode categorical features and scale numeric features.
5. Split the data into train and test sets with `test_size=0.2` and `random_state=42`.
6. Train multiple models.
7. Evaluate models with classification and ranking metrics.
8. Generate and save all visualizations in `outputs/`.
9. Select the best model automatically and explain why it performed best.

## Algorithms
The following supervised learning algorithms are trained and compared:
- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## Evaluation Metrics
Each model is evaluated using:
- Accuracy
- Precision
- Recall
- F1 Score
- ROC AUC
- Classification Report
- Confusion Matrix

## Results
The comparison table generated by the pipeline shows the following results:

| Model | Accuracy | Precision | Recall | F1 Score | ROC AUC |
| --- | ---: | ---: | ---: | ---: | ---: |
| Decision Tree Classifier | 0.5094 | 0.5025 | 0.5241 | 0.5131 | 0.5096 |
| Random Forest Classifier | 0.4821 | 0.4737 | 0.4500 | 0.4615 | 0.4907 |
| Logistic Regression | 0.5009 | 0.4926 | 0.4000 | 0.4415 | 0.4748 |

### Best Model
**Decision Tree Classifier**

It performed best because it achieved the highest F1 score and ROC AUC among the tested models. This suggests that the churn patterns in this dataset are better captured by rule-based splits than by a linear boundary or the current random forest baseline.
